# Math Operations

```java
int a = 20;
int b = 10; 

int result; 

result = a + b;  // 30 

result = a - b;  // 10 

result = a * b;  // 200 

result = a / b;  // 2 

result = a % b;  // 0
```

Basic math operations can be applied to `int`, `double` and `float` data types:

-   `+` addition
-   `-` subtraction
-   `*` multiplication
-   `/` division
-   `%` modulo (yields the remainder)

These operations are not supported for other data types.
